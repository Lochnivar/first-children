# Story Types

**Status:** Framework Element - Narrative Framework  
**Category:** Narrative / Story Types  
**Related:** [Character Types](character-types.md), [Themes](themes.md), [../FRAMEWORK.md](../../FRAMEWORK.md)

## Overview

The framework supports various story types that can be told within Hypatia. These story types provide structure while allowing for creative variation.

## Liberation Stories

### Core Focus
- Freeing worlds from cruel creator gods
- Hidden operations and careful planning
- Transferred souls as agents with outside perspective
- Breaking life force bindings safely

### Examples
- The Iolinus story (Nala vs. Zeta/Malachor)
- Other liberation operations on different worlds
- Variations with different First Children

### Key Elements
- Hidden operations necessary
- Gradual building of momentum
- High stakes (hostage situations)
- Moral complexity

## Creation Stories

### Core Focus
- First Children creating new worlds
- Following or violating the laws
- The process and consequences
- Origins and establishment

### Examples
- The creation of Iolinus (Zeta's violation)
- Legal world creation by First Children
- Origin stories and prequels
- Early universe stories

### Key Elements
- Creation process
- Decisions about binding
- Consequences of actions
- First Children motivations

## Inter-World Stories

### Core Focus
- Characters traveling between worlds
- Discovery of the multiverse
- Connections between different stories
- Cross-world characters and organizations

### Examples
- Characters discovering other worlds
- Organizations spanning multiple worlds
- Cross-world adventures
- Multiverse exploration

### Key Elements
- Travel mechanisms
- Discovery and revelation
- Connection building
- Expanded scope

## Other Conflicts

### Types
- First Children conflicts (beyond liberation)
- Mortal rebellions
- Discovery of quantum abilities
- Explorations of different worlds
- Internal struggles and growth

### Examples
- First Children disagreements
- Mortals rebelling against creators
- Characters discovering their abilities
- World exploration and discovery

### Key Elements
- Various conflict types
- Different scales and stakes
- Multiple perspectives
- Diverse motivations

## Story Variations

### Within Each Type
- Stories can vary significantly within each type
- Different tones, scales, and focuses
- Various character combinations
- Unique world-specific elements

### Combination Types
- Stories can combine multiple types
- Liberation with inter-world elements
- Creation with liberation aftermath
- Various hybrid approaches

## Guidelines

### Framework Compliance
- All story types must respect framework constants
- Can vary in details and approach
- Must work within framework rules
- Framework supports all story types

### Creative Freedom
- Significant creative freedom within types
- Framework provides structure, not limitation
- Unique approaches encouraged
- Variation is expected

## Notes

- Story types are flexible categories
- Framework supports diverse storytelling
- Types can overlap or combine
- Creativity thrives within structure
