# Slavic Cultural Template

**Type:** Major Culture Template  
**Description:** Eastern European cultural template  
**Era:** Hacutonce Era (HE)

## Overview

This directory contains the Slavic cultural template and its subcultures. The Slavic template serves as the foundational Earth-based cultural pattern for these related cultures.

## Subcultures

- [Zvezdan](subcultures/zvezdan/) - Nomadic type
- [Yaroslav](subcultures/yaroslav/) - Nomadic type

## Notes

- This is a major cultural template based on Earth cultural patterns
- Subcultures share foundational elements but have distinct characteristics
- Cultural indices are preserved from the original Azgaar export
