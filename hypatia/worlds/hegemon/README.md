# Hegemon

**Status:** Proposed World - Concept Development  
**Framework Conformance:** Active - See framework summary in [../hegemon.md](../hegemon.md)  
**Last Updated:** December 2024

## Overview

Hegemon is a living game board where multiple First Children test strategies of influence, control, and conquest through mortal civilizations. Created as a collaborative experiment in "controlled imbalance," the world has devolved into an ongoing competition among its creators. Mortals live real lives with real stakes while their destinies are shaped by godly manipulators playing geopolitical chess.

**Core Theme:** Power, strategy, and the moral cost of treating civilizations as game pieces.

## Framework Elements Demonstrated

- **Collaborative Creation:** Multiple First Children created Hegemon jointly
- **Ownership Conflicts:** Questions of "whose world is it?" drive competition
- **Peer Judgment:** External First Children debate whether this violates moralistic rules
- **Quantum Foundation:** All mechanics operate through quantum manipulation
- **Open Cosmology:** Mortals are aware of The Strategists (FC players)
- **Moral Complexity:** Explores the cost of treating mortals as game pieces
