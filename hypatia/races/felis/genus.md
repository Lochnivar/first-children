# Genus Felis

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [../README.md](../README.md)

## Definition

Genus Felis is defined by the cat-derived genetic template from the FC genetic repository. All Felis expressions—sapient and non-sapient—share these core traits:

### Physical
- **Predator template** — Sleek, agile, graceful
- **Enhanced senses** — Night vision, acute hearing
- **Silent movement** — Built for stealth

### Temperament
- **Solitary tendency** — Operate alone or in select company; pride in independence
- **Pride** — Reputation-focused; perfection matters (expressed differently by species)
- **Patient** — Wait for the right moment

### Framework
- **Non-sapient baseline:** Cats (Earth template)
- **Sapient expression:** [Nekari](nekari.md) (*Felis praecisus*) — perfection of purpose or action; precision work, solitude
- **Sapient expression:** [Felari](felari.md) (*Felis elegans*) — perfection of appearance or manners; social grace, presentation
