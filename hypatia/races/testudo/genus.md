**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [../README.md](../README.md)

## Definition

Genus Testudo is defined by the tortoise-derived genetic template from the Prime Author's repository. All Testudo expressions—sapient and non-sapient—share these core traits:

### Physical
- **Shell-backed** — Protective carapace; can retreat into shell
- **Long-lived** — Tortoise-derived lifespan (centuries)

### Temperament
- **Patient** — Deeply patient, methodical
- **Slower of movement, temperament, and thought** — Deliberate pace in all things; calm, unhurried

### Framework
- **Non-sapient baseline:** Tortoises (Earth template)
- **Sapient expressions:**
  - [Chelonari](chelonari.md) — traits tend toward helping others with wisdom
  - [Aubrei](aubrei.md) — naval-themed; more aggressive; same core traits
