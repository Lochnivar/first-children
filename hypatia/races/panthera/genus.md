# Genus Panthera

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [../README.md](../README.md)

## Definition

Genus Panthera is defined by the big-cat-derived genetic template from the FC genetic repository. All Panthera expressions—sapient and non-sapient—share these core traits:

### Physical
- **Predator template** — Sleek, agile, deadly
- **Enhanced senses** — Night vision, acute hearing
- **Silent movement** — Built for stealth

### Temperament
- **Solitary** — Operate alone; pride in independence
- **Professional** — Reputation-focused, no wasted emotion
- **Patient hunters** — Wait for the right moment

### Framework
- **Non-sapient baseline:** Big cats (Earth template)
- **Sapient expression:** [Panthers](panthers.md) — traits tend toward precision work, solitude
