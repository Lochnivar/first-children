# Races of Hypatia

**Status:** Framework Element - Universal  
**Category:** Races / Species  
**Related:** [../FRAMEWORK.md](../../FRAMEWORK.md)

## Overview

This directory contains documentation for intelligent races/species that exist in Hypatia. These races may or may not appear on individual worlds, but they are available as options for world-building across the multiverse.

## Universal Races

### Core Races
- **[Humans](humans.md)** - The baseline intelligent species, present on many worlds
- **[Elves](elves.md)** - Long-lived, graceful, often connected to nature or magic
- **[Dwarves](dwarves.md)** - Stout, hardy, often master craftspeople
- **[Orcs](orcs.md)** - Strong, warrior-focused, varied cultural expressions
- **[Halflings](halflings.md)** - Small, adaptable, community-oriented

### Additional Races
- **[Kitsune](kitsune.md)** - Shapeshifting fox-people, intelligent tricksters with magical abilities
- **[Giff](giff.md)** - Hippo-like humanoids, military-focused with love of explosives

### Other Races (To Be Defined)
- Additional fantasy races as needed
- Unique races specific to individual worlds

## Usage Across Worlds

### World-Specific Implementation
- Individual worlds may include all, some, or none of these races
- Races may manifest differently on different worlds
- Each world may have unique races not in this framework
- Framework races provide common options for consistency

### Iolinus Status
- Iolinus may or may not include these races
- Current Iolinus documentation focuses on human cultures
- Framework races are available for future stories or other worlds

## Documentation Structure

Each race file includes:
- Universal characteristics
- Common variations
- Framework-level description
- Notes on how they might vary across worlds

## Guidelines

- Framework races are universal options
- Worlds can include, exclude, or modify races
- Worlds can add unique races not in framework
- Framework provides common baseline for consistency
