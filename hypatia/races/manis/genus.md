# Genus Manis

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [../README.md](../README.md)

## Definition

Genus Manis is defined by the pangolin-derived genetic template from the FC genetic repository. All Manis expressions—sapient and non-sapient—share these core traits:

### Physical
- **Chitinous plates** — Overlapping scales; full body coverage; can curl into armored sphere
- **Compact build** — Low center of gravity; optimized for defensive curling
- **Long-lived** — Pangolin-derived lifespan (centuries)

### Temperament
- **Patient** — Deeply patient; minimal energy expenditure
- **Deliberate** — Slow movement, minimal speech; no wasted motion

### Framework
- **Non-sapient baseline:** Pangolins (Earth template)
- **Sapient expressions:**
  - [Quedari](quedari.md) — Manis sapiens; survived trial via passive avoidance
