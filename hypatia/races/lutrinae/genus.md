# Genus Lutrinae

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [Races README](../README.md)

## Definition

Genus Lutrinae is defined by the otter-derived genetic template from the FC genetic repository. No designated role; traits lead to emergent facilitation, bridge-building, and—in some individuals—grift. Charm and play as tools; warmth as bond and leverage.

### Physical
- **Otter-derived** — Small ears, whiskers, expressive; webbed digits
- **Compact** — 4–5 feet tall; lean, agile
- **Tool use** — Natural; carry favorite objects (rocks, stones)

### Temperament
- **Play-as-bonding** — Build trust through shared play; process stress through silliness
- **Charm-aware** — Know they're disarming; deploy it deliberately
- **Variable** — Some facilitators, some grifters, some both; you never know which

### Framework
- **Non-sapient baseline:** Otters (Earth template)
- **Sapient expression:** [Otter-folk](otter-folk.md) — bridge-builders, facilitators, grifters; no designated role
