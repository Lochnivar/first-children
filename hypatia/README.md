# Framework Documentation

**Detailed documentation of the Hypatia Framework.**

Hypatia establishes the universal rules, cosmology, and metaphysics that govern all worlds in Hypatia.

**Conceptual Metaphor: The Library (Purely Conceptual)**
- **Hypatia** is conceptually understood as "The Library" containing all existence—purely conceptual, the closest approximation First Children or mortals can achieve
- **Worlds** are conceptually understood as "Books" within the library, each telling their own story—a mental model, not literal structure
- **First Children** conceptually "inhabit the library" and can "access the books/worlds"—this metaphor helps them organize their understanding
- **Big 'G' God** is conceptually understood as "The Author" who created the library and established its rules 

## Framework vs. Worlds

**The Framework = General/Universal Rules**
- Contains the "all or general" things that apply across the entire over-universe
- Establishes universal constants and fundamental principles
- Provides the foundational structure for all worlds

**World Details = Expansions or Exceptions**
- **Expansions:** World-specific details that add more depth and specificity within the framework rules
  - Example: Framework says "mortals may have fragmented quantum abilities" → Iolinus expands this with specific five abilities
  - Example: Framework establishes soul transfer mechanics → Iolinus expands with greyrose-specific implementation
- **Exceptions:** World-specific variations that work within or modify general rules
  - Example: Framework establishes universal principles → World may have unique cultural interpretations
  - Example: Framework provides patterns → World demonstrates a variation of those patterns

All individual stories and worlds must conform to the framework rules, but they expand upon or provide exceptions to those general rules through their unique details.

## Framework Overview

The Hypatia framework establishes:

### Core Cosmological Structure
- **Big 'G' God** - The deistic creator who established the laws
- **First Children** - Ancient immortal beings with Quantumancy
- **Multiple Worlds** - Worlds created by First Children
- **Universal Forces** - Quantumancy, soul transfer, life force binding

### Rules and Guidelines
- What must remain consistent across all stories
- What can vary between worlds
- How to expand while maintaining consistency
- Connection points between worlds

### Story Patterns
- Liberation stories (freeing worlds from creators)
- Creation stories (First Children creating worlds)
- Inter-world stories (travel between worlds)
- Character types and themes

## Working with the Framework

1. **Framework is PRIMARY** - The framework is the source of truth
2. **Check First** - Before making changes, check framework rules
3. **Document Conformance** - Each world should document how it conforms
4. **Process Contradictions** - See [../meta/CONSISTENCY_PROCESS.md](../meta/CONSISTENCY_PROCESS.md)

## Key Concepts

- **Quantumancy** - Full quantum manipulation (First Children only)
- **Quantum Abilities** - Fragmented versions in mortals (varies by world)
- **Soul Transfer** - Mechanism for moving souls between worlds
- **Protective Mechanisms** - Worlds may have protective mechanisms (life force binding is Iolinus-specific)
- **Prohibitions** - Laws established by Big 'G' God that First Children may violate
