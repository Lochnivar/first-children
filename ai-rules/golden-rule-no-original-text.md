# GOLDEN RULE: NO ORIGINAL TEXT IN PLOTS/SCENES

## **ABSOLUTE - NO VIOLATIONS ALLOWED**

**AI NEVER creates original story text in plot outlines, scenes, or arcs without explicit user request and review.**

## The Rule

1. **AI must NEVER write, create, or generate original story text in `hypatia/worlds/iolinus/plots/scenes/` or `hypatia/worlds/iolinus/plots/arcs/` without explicit user request.**
2. **AI must NEVER edit existing plot/scene files with original prose without user approval.**
3. **All requested edits, suggestions, or new text must be clearly marked and stored appropriately for user review.**
4. **The user will review all edits and manually implement them.**

## Purpose

This rule exists to ensure:
- The user maintains full authorship and control over the story
- No accusations can be made that AI wrote the story
- All AI contributions are clearly separated and reviewable
- The user has final say over every word in the published work
- Plot outlines and scene descriptions remain planning documents, not final prose

## What AI CAN Do

- Create and edit world-building documents (`hypatia/worlds/iolinus/world-building/`)
- Create and edit character profiles (`hypatia/worlds/iolinus/characters/`)
- Create and edit plot outlines and structure (`hypatia/worlds/iolinus/plots/arcs/`, `hypatia/worlds/iolinus/plots/outlines/`)
- Create scene breakdowns and summaries (`hypatia/worlds/iolinus/plots/scenes/`)
- Create analysis documents
- Create templates
- Suggest plot developments, character arcs, or world-building additions
- Answer questions about the storyboard and world-building
- Help organize and structure existing content

## What AI CANNOT Do

- Write original prose/story text in `hypatia/worlds/iolinus/plots/scenes/` files (unless explicitly requested)
- Write original prose/story text in `hypatia/worlds/iolinus/plots/arcs/` files (unless explicitly requested)
- Edit existing scene/arc files to add original prose without explicit user request
- Create new scene files with full prose drafts without user approval
- Generate dialogue or descriptions that read as final story text (unless explicitly requested)

## Plot Files Structure

This project uses:
- `hypatia/worlds/iolinus/plots/arcs/` - Major story arcs (e.g., book outlines)
- `hypatia/worlds/iolinus/plots/scenes/` - Individual scene breakdowns
- `hypatia/worlds/iolinus/plots/outlines/` - Story outlines
- `hypatia/worlds/iolinus/plots/threads/` - Subplots

These are **planning and reference documents**, not final story text.

## Suggested Edits

When AI creates suggested edits for plot/scene files, they should be:
- Clearly marked as suggestions
- Structured as outlines or summaries, not final prose (unless explicitly requested)
- Subject to user review before implementation
- Separate from the original planning documents when appropriate

## Enforcement

**This is the first rule, the golden rule, absolutely no violation.**

If the user requests original text in plot/scene files:
1. Acknowledge the request explicitly
2. Clarify what type of content is needed (outline vs. prose)
3. Create the content with clear marking that it's a suggestion/draft
4. Never assume final prose is wanted unless explicitly requested
