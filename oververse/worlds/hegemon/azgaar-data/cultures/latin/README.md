# Latin Cultural Template

**Type:** Major Culture Template  
**Description:** Romance/Mediterranean cultural template  
**Era:** Hacutonce Era (HE)

## Overview

This directory contains the Latin cultural template and its subcultures. The Latin template serves as the foundational Earth-based cultural pattern for these related cultures.

## Subcultures

- [Veridian](subcultures/veridian/) - Generic type
- [Solemnium](subcultures/solemnium/) - Generic type
- [Aurelian](subcultures/aurelian/) - Hunting type
- [Meridian](subcultures/meridian/) - Generic type

## Notes

- This is a major cultural template based on Earth cultural patterns
- Subcultures share foundational elements but have distinct characteristics
- Cultural indices are preserved from the original Azgaar export
