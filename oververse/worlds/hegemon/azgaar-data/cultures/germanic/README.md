# Germanic Cultural Template

**Type:** Major Culture Template  
**Description:** Northern European cultural template  
**Era:** Hacutonce Era (HE)

## Overview

This directory contains the Germanic cultural template and its subcultures. The Germanic template serves as the foundational Earth-based cultural pattern for these related cultures.

## Subcultures

- [Valthir](subcultures/valthir/) - Generic type
- [Sturmgaard](subcultures/sturmgaard/) - Generic type
- [Eldermark](subcultures/eldermark/) - Generic type

## Notes

- This is a major cultural template based on Earth cultural patterns
- Subcultures share foundational elements but have distinct characteristics
- Cultural indices are preserved from the original Azgaar export
