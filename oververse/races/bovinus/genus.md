# Genus Bovinus

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [Races README](../README.md)

## Definition

Genus Bovinus is defined by the bovine/cattle-derived **non-sapient** genetic template from the FC genetic repository (cattle, oxen, bison). Sapient bull-derived **staff** are not placed here—they are [*Homo bovinus*](../homo/bovinus.md) under Genus Homo (FC purpose-built security and shock troops).

### Physical
- **Bovine-derived** — Bull or ox framework; horns, heavy build, powerful
- **Large** — 7–9 feet tall; 400–700 lbs
- **Strong** — Built for sustained labor, not burst speed

### Temperament
- **Structural** — Prefer clear paths, direct routes, logical organization
- **Patient** — Can work long hours; endurance over speed
- **Deliberate** — Think before acting; prefer certainty to improvisation
- **Territorial (differently)** — Not aggressive; more "this is the way it's built"

### Framework
- **Non-sapient baseline:** Cattle, oxen, bison (Earth template)
- **Sapient expression:** None in this genus. Bovine-derived FC staff are *Homo bovinus* — see [Homo bovinus](../homo/bovinus.md).
