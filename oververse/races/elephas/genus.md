# Genus Elephas

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [../README.md](../README.md)

## Definition

Genus Elephas is defined by the elephant-derived genetic template from the FC genetic repository. All Elephas expressions—sapient and non-sapient—share these core traits:

### Physical
- **Large, strong build** — Massive frame, tank-like
- **Long-lived** — Elephant-derived lifespan (centuries)
- **Incredible memory** — Exceptional recall

### Temperament
- **Philosophical, contemplative** — Deep thinkers, melancholic wisdom
- **Social, herd-oriented** — Strong bonds, communal
- **Wanderlust** — Neurological compulsion to travel (sapient expression amplifies this)

### Framework
- **Non-sapient baseline:** Elephants (Earth template)
- **Sapient expression:** [Pachi](pachi.md) — traits tend toward movement, trade, contemplation
