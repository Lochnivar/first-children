# Genus Capra

**Status:** Framework Element - Universal  
**Category:** Races / Genus Definition  
**Related:** [Genetic Frameworks](../genetic-frameworks.md), [Races README](../README.md)

## Definition

Genus Capra is defined by the goat-derived genetic template from the FC genetic repository. All Capra expressions share core traits: sure-footed, stubborn, territorial, hardy.

### Physical
- **Hooved, digitigrade** — Sure-footed, perfect balance
- **Horned** — Various sizes; some expressions have quantum-sensitive horns
- **Rectangular pupils** — Horizontal slits; excellent peripheral vision
- **Climber's physique** — Lean, wiry, agile

### Temperament
- **Stubborn** — Immovable once positioned
- **Territorial** — Fiercely protective of assigned space
- **Independent** — Self-directed, minimal supervision
- **Hardy** — Work through fatigue, stoic

### Framework
- **Non-sapient baseline:** Goats (Earth template)
- **Sapient expression:** [Kethi (*Capra Quantus*)](quantus.md) — panspermia species with quantum-sensitive horns; **Riftkeepers** are a subset employed by Virelle at Mercara's Cross
