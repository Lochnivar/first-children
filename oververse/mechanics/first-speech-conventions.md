# First Speech conventions (trials & quest interfaces)

**Status:** Framework element — universal (cross-world)  
**Category:** Mechanics / naming / UI  
**Related:** [Trials](trials.md), [FRAMEWORK.md](../../FRAMEWORK.md)

## Disambiguation

**This document is not** [First Speech (cosmology)](../cosmology/first-speech.md), which names the Prime Author’s original conceptual framework (*hypatia*, foundational vocabulary).

**This document is** about a **recurring in-universe naming pattern**: how First Children **trials**, **quest OSes**, and **deployment branding** label worlds and products using **cryptic, “First Speech–flavored” strings**—the same *kind* of orthographic habit everywhere participants are dropped. In-setting, the label **“First Speech”** is often applied to this pattern (marketing, engineer slang, or bureaucratic habit); it **does not** assert sacred or Prime-Author theology for those strings unless a story explicitly ties them.

---

## What the convention is

Across Hypatia deployments that use **instrumented trials** (LitRPG-style interfaces, orientation quests, wetware HUDs), naming tends to follow a **shared aesthetic**:

- **World / deployment tokens** — A short, often awkward string that **identifies which reality** the questing stack is bound to (e.g. orientation headers, instance IDs). Not the world’s common name in local language; the **system’s** name for that book.
- **Product / voice layer names** — Shared shells (e.g. **iQuest** and its **First Speech for voice**, documented under stack terminology) that can appear **across** multiple worlds.
- **Orthographic quirks** — Leading apostrophes, leading grave accents (backticks), compressed consonant clusters, strings that scan as **almost pronounceable** but **wrong**—**phonetic estrangement**: alien enough to feel off, close enough to feel patterned.

This is a **global trope** in-setting: readers and participants learn to recognize “that kind of name” before they learn what it stands for.

## Layers (how pieces stack)

| Layer | Role | Typical feel |
|--------|------|----------------|
| **Network / plumbing** | Carries signals (e.g. quantum/wetware transport) | Technical acronyms, “handshake,” low-level readouts |
| **Quest / trial product** | Presents rules, quests, character generation | Product names (e.g. **iQuest**) |
| **Voice / UI “First Speech”** | Participant-facing snark, bullets, policy | Branded strings (e.g. **QWST** with leading backtick as First Speech for iQuest) |
| **World token** | **Which** deployment | Per-world cryptic label (see table below) |

Concrete stack terminology (QWAN, `` `QWST``, iQuest) lives in the Algernon world bible as a worked example: [QWAN, QWST, iQuest, First Speech](../worlds/algernon/world-building/qwan-qwst-first-speech.md).

## World tokens (examples)

Canonical examples of **First Speech for [world]** as a **deployment label**:

| World | Token | Typical use |
|--------|--------|----------------|
| **Algernon** | `'LGRNN` | e.g. *'LGRNN Orientation Quest* — leading apostrophe is part of the string |
| **Atahom** | `'THM` | Deployment label; common **Tahom**; extended roots & locative rules: [First Speech — Atahom](../worlds/atahom/world-building/first-speech-lexicon.md) |
| **Legara** | `LGRH` | First Speech for Legara |
| **Mercara** | `MRCRH` | First Speech for Mercara |

New worlds can add rows here as they enter trial fiction.

## HUD and register (optional patterns)

Fiction may distinguish **official** system lines from **unofficial bleed** (e.g. `*` vs `~` in fenced UI), as in [TheWay Interface](../worlds/algernon/world-building/theway-interface.md). Not every world uses the same glyph rules; Algernon is the reference implementation for **Charlie**-era prose.

---

## Design intent (author-facing)

- **Phonetic / structural echo** of some natural-language families (e.g. Semitic-like patterning) is allowed as **texture**—**odd but recognizable** to many English readers.
- **Explicitly not** the goal: biblical narrative, sacred-text allegory, or dressing real-world religion as set dressing.
- **Cosmology vs. trials:** Keep [Prime Author First Speech](../cosmology/first-speech.md) and **trial naming** mentally separate unless a story **merges** them on purpose.
