---

## Convention

- **Root system:** Semitic consonantal roots (Hebrew/Aramaic tradition) with vowel-filling to produce pronounceable names
- **Meaning:** Informs but does not constrain — FC drift far from their name's root meaning over time
- **Key principle:** Categories are starting points, not definitions. A FC named from the Shadow cluster is not automatically antagonistic. A FC named from the Regal cluster is not automatically authoritative. Names reflect origin orientation, not destiny.

## Usage Notes

**Canonical vs. Suggested:** Names in the first table are established. Names in the pool above are suggestions — a working set to draw from, not an exhaustive or fixed list.

**For new FC development:**
1. Identify the FC's foundational orientation — what they were built around before personality and experience shaped them
2. Select a cluster that matches that orientation
3. Choose a root whose meaning resonates
4. Fill vowels for pronounceability — the name should land cleanly when spoken aloud
5. Note that meaning informs origin, not destiny — the FC will drift

**Reserved / Handle With Care:**
- THWM (Tehom) — primordial weight, use for significant cosmological FC only
- 'WR (Or) — foundational illumination, similarly weighty
- GLGL (Galgal) — trial architect potential, strong thematic resonance
- RBQH (Rivqah) — antagonist potential distinct from Malachor
- 'DM (Adam) — Homo genus resonance, use with intention

**Currently Unused Clusters:**
- Shadow/Decoherence — no canonical FC yet
- Process/Transformation — new cluster, no canonical FC yet

**Other naming traditions:**
- **Thalor Voss** — Vibratory/Resonant (boundary curator). Non-root naming. Co-sponsors Protomundus with Virelle in MC20.
- **Zorathar** (The Architect) — Delverium creator. Meritocracy experiment. MC20 World 7 sponsor.
- **Keyes** — Algernon creator/operator. Phase transition experiment. MC20 World 14 sponsor.
